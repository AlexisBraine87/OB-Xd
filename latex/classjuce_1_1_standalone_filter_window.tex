\doxysubsection{Detailed Description}
A class that can be used to run a simple standalone application containing your filter.

Just create one of these objects in your JUCEApplication\+Base\+::initialise() method, and let it do its work. It will create your filter object using the same \doxylink{_plugin_processor_8cpp_a6a42963b424a437ac872809903afa10e}{create\+Plugin\+Filter()} function that the other plugin wrappers use.
